## Database schema

## How to run
> `npm install`
> `npm start`

## Configurations

Required the following env keys

| Key                                       | Default |
| ----------------------------------------- | ------- |
| `SERVER_PORT`                         | 3000    |
| `CORS_ORIGIN`                             | '*'    |
| `PASSWORD_SALT_ROUNDS`                    | 10      |
| `ACCESS_TOKEN_EXPIRY`                     |    30m   |
| `ACCESS_TOKEN_SECRET`                     |    TEST  |
| `DB_USERNAME`                   |         |
| `DB_PASSWORD`                   |         |
| `DB_PORT`                       |         |
| `DB_DATABASE`                   |         |
| `DB_HOST`                       |         |
| `NODE_ENV`                                |         |
